🎬 CinemaVault

A simple and elegant Command Line Interface (CLI) application to manage your personal movie diary.
Track what you watch, where you watched it, and your thoughts — all in one place.



📌 Features
	•	➕ Add new movie entries
	•	📖 View all saved entries
	•	💬 Store personal reviews/thoughts
	•	🌐 Track platform (Netflix, Theatre, etc.)
	•	💾 Persistent storage using JSON



🛠️ Tech Stack
	•	Python
	•	JSON (for data storage)
	•	CLI (Command Line Interface)



📂 Project Structure

cine-log-cli/
│── cinema_diary_cli.py
│── cinema_diary.json
│── README.md




▶️ How to Run
	1.	Clone the repository:

git clone https://github.com/your-username/cine-log-cli.git

	2.	Navigate to the project folder:

cd cine-log-cli

	3.	Run the application:

python cinema_diary_cli.py



🧠 How It Works
	•	The app stores movie entries in a JSON file (cinema_diary.json)
	•	Users can:
	•	Add a movie with title, language, platform, and thoughts
	•	View all stored entries in a structured format
	•	Data persists between runs
